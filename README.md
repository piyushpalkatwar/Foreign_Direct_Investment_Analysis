
#  Foreign Direct Investment (FDI) Analysis Project

##  Project Overview

This project focuses on analyzing **Foreign Direct Investment (FDI)** trends using Python and real-world investment data.

The objective of the project is to study how foreign investments are distributed across different sectors, identify investment growth patterns, understand sector-wise contribution, and generate meaningful economic insights through data analysis and visualization.

This project demonstrates strong **data analysis, financial analytics, visualization, and business insight generation** skills.

---

##  Business Problem

Foreign Direct Investment plays a critical role in economic development. Governments and businesses need to understand:

* Which sectors receive the highest foreign investment
* Investment growth trends over time
* Sector-wise contribution in total investment
* Economic sectors attracting global investors
* Areas with declining investment performance

This project solves these questions through exploratory data analysis.

---

##  Project Objectives

Main objectives of this project:

* Analyze yearly FDI investment trends
* Study sector-wise investment contribution
* Identify top sectors receiving highest FDI
* Compare investment distribution across industries
* Detect sectors with consistent growth
* Generate business insights from investment data
* Perform economic trend analysis using real-world dataset
